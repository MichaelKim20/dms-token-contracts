## Install NodeJS

https://nodejs.org/en/download

## Install yarn

```shell
npm install -g yarn
```

## Install Project

```shell
git clone https://github.com/acc-coin/acc-token-contracts.git
cd acc-token-contracts
yarn install
```

## Test Project

```shell
cp -r env/.env.sample env/.env
yarn run test
```
